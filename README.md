# easy-flux-example
An easy React application that demonstrates basic usage of the Flux architecture pattern.  For more info on Flux, visit the [official Flux website](https://facebook.github.io/flux/).

## Tutorial
I've set this up as a step-by-step tutorial on my blob.  [Click here to follow the tutorial on building this Easy Flux Example](http://tonyspiro.com/an-easy-flux-example-application/).
##Getting started
```
git clone https://github.com/tonyspiro/easy-flux-example
```

```
cd easy-flux-example
```

```
npm install
```

Optional:

Gulp
```
gulp watch
```
HTTP Server
```
sudo npm install -g http-server
http-server
```

Go to http://localhost:8080
